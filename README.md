
# C# Bootcamp

C# bootcamp consists of topics that cover the fundamentals of the C# programming language.


# Objective

- Understand the fundamentals of Object-Oriented Programming (OOP).
- Learn how to create and manage classes, objects, and methods.
- Knowledge of C# features like managed and unsafe code.
